### Файлы:

## Инструкции по применению изменений

### 1. Перезапустите backend
```bash
cd backend
npm run start:dev
```

TypeORM автоматически создаст новое поле `password` в таблице `users`.

### 2. Обновите существующих пользователей (опционально)

Если у вас уже есть пользователи в базе данных, выполните скрипт:

```bash
cd backend
node update-users-password.js
```

Этот скрипт установит временный пароль `password123` для всех существующих пользователей.

⚠️ **Важно**: После этого пользователи должны будут войти с паролем `password123` и сменить его в настройках профиля.

### 3. Новые пользователи

Новые пользователи, которые зарегистрируются после обновления, будут иметь нормальные пароли, которые они указали при регистрации.

✅ Проверка наличия товара перед оформлением заказа
