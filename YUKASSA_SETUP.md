# Настройка ЮKassa для приема платежей

## ✅ Что уже сделано:

1. **Backend модуль платежей** - полностью готов
2. **Внутренний кошелек** - работает
3. **Интеграция с ЮKassa API** - реализована
4. **Страница Финансы** - полностью функциональна
5. **Поддержка СБП и карт** - через ЮKassa

## 🚀 Как запустить в тестовом режиме:

### Шаг 1: Регистрация в ЮKassa

1. Перейдите на https://yookassa.ru/
2. Нажмите "Подключить ЮKassa"
3. Зарегистрируйтесь (нужны данные ИП или ООО, но для теста можно использовать тестовый магазин)

### Шаг 2: Получите тестовые ключи

1. Войдите в личный кабинет ЮKassa
2. Перейдите в раздел "Настройки" → "Данные для API"
3. Скопируйте:
   - **shopId** (идентификатор магазина)
   - **Секретный ключ**

### Шаг 3: Добавьте ключи в .env

Откройте `backend/.env` и добавьте:

```env
YOOKASSA_SHOP_ID=ваш_shop_id
YOOKASSA_SECRET_KEY=ваш_секретный_ключ
```

### Шаг 4: Перезапустите backend

```bash
cd backend
npm run start:dev
```

## 🧪 Тестирование

### Тестовые карты ЮKassa:

**Успешная оплата:**
- Номер: `5555 5555 5555 4444`
- Срок: любая будущая дата
- CVC: любые 3 цифры

**Отклоненная оплата:**
- Номер: `5555 5555 5555 5599`

**3-D Secure:**
- Номер: `4111 1111 1111 1111`
- Код подтверждения: любой

### Тестирование СБП:

В тестовом режиме СБП работает через эмуляцию - просто выбирайте любой банк.

## 💰 Как это работает:

### Пополнение баланса:

1. Пользователь заходит в "Финансы"
2. Нажимает "Пополнить баланс"
3. Выбирает способ (карта или СБП)
4. Вводит сумму (минимум 10 ₽)
5. Перенаправляется на страницу оплаты ЮKassa
6. После оплаты возвращается обратно
7. Баланс автоматически пополняется

### Оплата заказа с баланса:

1. При оформлении заказа выбирает "Оплатить с баланса"
2. Если средств достаточно - заказ оплачивается мгновенно
3. Средства списываются с кошелька

## 📊 Что доступно на странице Финансы:

### Вкладка "Обзор":
- Текущий баланс кошелька
- Всего пополнено
- Всего потрачено
- Количество транзакций
- Последние 5 транзакций
- Кнопка "Пополнить баланс"

### Вкладка "История транзакций":
- Полная история всех операций
- Фильтры по типу и статусу
- Детальная информация о каждой транзакции

### Вкладка "Пополнение баланса":
- Выбор способа оплаты (карта/СБП)
- Быстрые суммы (100, 500, 1000, 5000 ₽)
- Информация о безопасности

## 🔧 Структура базы данных:

### Таблица `wallets`:
- `userId` - ID пользователя
- `balance` - текущий баланс
- `totalDeposited` - всего пополнено
- `totalSpent` - всего потрачено

### Таблица `transactions`:
- `userId` - ID пользователя
- `amount` - сумма
- `type` - тип (DEPOSIT, PAYMENT, REFUND, WITHDRAWAL)
- `status` - статус (PENDING, COMPLETED, FAILED, CANCELLED)
- `paymentMethod` - способ оплаты
- `externalId` - ID платежа в ЮKassa
- `description` - описание

## 🎯 API Endpoints:

```
GET  /payments/wallet/:userId          - Получить кошелек
GET  /payments/transactions/:userId    - Получить транзакции
POST /payments/deposit                 - Создать платеж для пополнения
POST /payments/pay-order               - Оплатить заказ с баланса
POST /payments/callback                - Webhook от ЮKassa
```

## 💡 Советы:

1. **Для разработки** используйте тестовые ключи
2. **Для продакшена** получите боевые ключи и пройдите модерацию
3. **Webhook URL** настройте в личном кабинете ЮKassa: `https://ваш-домен.ru/payments/callback`
4. **Минимальная сумма** пополнения - 10 ₽ (можно изменить в коде)
5. **Комиссия ЮKassa** - обычно 2.8-3.5% (в тестовом режиме - 0%)
