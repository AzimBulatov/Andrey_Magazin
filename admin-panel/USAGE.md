# Инструкция по использованию админ-панели

## Запуск

```bash
npm install
npm run dev
```

Админ-панель будет доступна по адресу: http://localhost:5173

## Функционал

### 1. Категории

1. Перейдите на вкладку "Категории"
2. Нажмите "+ Добавить категорию"
3. Заполните форму:
   - Название (обязательно)
   - Описание (опционально)
   - Активна (чекбокс)
4. Нажмите "Создать"

### 2. Товары

**Важно:** Сначала создайте хотя бы одну категорию!

1. Перейдите на вкладку "Товары"
2. Нажмите "+ Добавить товар"
3. Заполните форму:
   - Название (обязательно)
   - Описание (опционально)
   - Цена в рублях (обязательно)
   - Количество на складе (обязательно)
   - Категория (выберите из списка)
   - Изображение (опционально, до 5MB)
   - Активен (чекбокс)
4. Нажмите "Создать"

### 3. Редактирование

- Нажмите на кнопку "✏️" для редактирования
- Измените нужные поля
- Нажмите "Сохранить"

### 4. Удаление

- Нажмите на кнопку "🗑️" для удаления
- Подтвердите действие

## Устранение проблем

### Товар не создается

1. Убедитесь, что создана хотя бы одна категория
2. Проверьте, что все обязательные поля заполнены
3. Проверьте консоль браузера (F12) на наличие ошибок
4. Убедитесь, что бэкенд запущен на порту 3000

### Изображение не загружается

1. Проверьте размер файла (максимум 5MB)
2. Используйте форматы: JPG, PNG, GIF, WEBP
3. Убедитесь, что папка `backend/uploads` существует и доступна для записи

### Модальное окно не по центру

Обновите страницу (Ctrl+R) - исправление уже применено.
