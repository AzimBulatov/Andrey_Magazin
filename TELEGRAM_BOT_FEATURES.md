# Новые возможности Telegram бота

## 🌐 Вход на сайт через Telegram

Теперь пользователи могут войти на сайт прямо из Telegram бота без необходимости регистрации и ввода пароля!

### Как использовать:

1. **В главном меню бота** нажмите кнопку "🌐 Войти на сайт"
2. **Или введите команду** `/website`
3. Бот отправит вам **персональную ссылку**
4. **Нажмите на ссылку** - вы автоматически войдете на сайт
5. Ссылка действует **10 минут** и **одноразовая**

### Преимущества:

✅ Не нужно запоминать пароль  
✅ Быстрый вход в один клик  
✅ Безопасно - ссылка одноразовая  
✅ Все ваши заказы и данные сохраняются

## ℹ️ Обновленный раздел "Помощь"

Раздел помощи теперь содержит подробные ответы на часто задаваемые вопросы:

### Доступные темы:

#### ❓ Как сделать заказ?
Пошаговая инструкция по оформлению заказа в боте

#### 💳 Способы оплаты
Информация о доступных методах оплаты:
- Наличными курьеру
- Картой при получении
- Онлайн-оплата (скоро)

#### 🚚 Доставка
Условия доставки:
- Сроки: 1-3 рабочих дня
- Бесплатная доставка от 5000 ₽
- Курьер свяжется для уточнения времени

#### ↩️ Возврат товара
Правила возврата:
- В течение 14 дней
- С сохраненными бирками
- Возврат денег за 3-5 дней

#### 👤 Мой аккаунт
Информация об управлении аккаунтом и входе на сайт

#### 📞 Связаться с нами
Контактная информация службы поддержки

## Технические детали

### Для разработчиков:

**Backend:**
- Новая сущность `TelegramAuthToken` для хранения токенов
- Эндпоинты для генерации и валидации токенов
- Автоматическая очистка истекших токенов

**Frontend:**
- Новая страница `/auth/telegram` для обработки авторизации
- Автоматическое сохранение JWT токена
- Перенаправление на главную после успешного входа

**Безопасность:**
- Токены действуют 10 минут
- Одноразовое использование
- Привязка к Telegram ID
- Автоматическое удаление старых токенов
