## 🧪 Как протестировать

### 1. Запустите приложение:

```bash
# Backend
cd backend
npm run start:dev

# Frontend (в другом терминале)
cd client-site
npm run dev
```

### 2. Откройте Telegram бот

### 3. Попробуйте новые функции:

**Вход на сайт:**
- Нажмите "🌐 Войти на сайт" в главном меню
- Или введите `/website`
- Нажмите на полученную ссылку
- Вы автоматически войдете на сайт!

**FAQ:**
- Нажмите "ℹ️ Помощь" в главном меню
- Выберите интересующую тему
- Прочитайте подробный ответ

## 🔧 Настройка для продакшена

1. **Обновите `.env`:**
```env
WEBSITE_URL="https://yourdomain.com"
```

2. **Настройте HTTPS** для вашего домена

3. **Настройте CORS** в backend для вашего домена

4. **Отключите synchronize** в TypeORM:
```typescript
synchronize: false  // в app.module.ts
```

5. **Создайте миграции** для таблицы `telegram_auth_tokens`

## ❓ Частые вопросы

**Q: Ссылка не работает**  
A: Ссылка действует только 10 минут и одноразовая. Запросите новую в боте.

**Q: Пользователь не найден**  
A: Пользователь должен хотя бы раз написать боту, чтобы создался аккаунт.

**Q: Как изменить время действия токена?**  
A: В `auth.service.ts` измените строку:
```typescript
expiresAt.setMinutes(expiresAt.getMinutes() + 10); // 10 минут
```

**Q: Как настроить FAQ?**  
A: В `bot.update.ts` найдите метод `handleFAQ` и отредактируйте объект `faqContent`.
